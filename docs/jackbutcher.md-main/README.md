# jackbutcher.md

50,000 tweets. One file. Every pattern extracted.

## What this is

6 years of writing. 50,796 tweets scanned. Filtered to the 13,962 that performed. Reverse-engineered into the rules, constraints, and word-level mechanics that made them work.

Drop it into any AI and generate content in this voice.

## What's extracted

- **6 rhetorical moves:** contrast pairs, reframes, math, uncomfortable truths, compressed wisdom, deadpan humor
- **11 word-level mechanics:** alliterative contrasts, monosyllabic endings, chiasmus, circular loops, negation flips, and more
- **12 contrast frames:** ranked by frequency with examples
- **Statistical profile:** median 9 words, 57% under 10, 78% single sentence
- **Anti-patterns:** what the voice never does (no hedging, no self, no diary, no cliches)

## How to use it

Drop the file into any AI conversation as context. Say "write in this voice." The output won't be perfect — but it will sound like the person, not a chatbot.

The file isn't the moat. The person is. Anyone can use this file to generate content in this voice. Nobody can generate the six years of daily practice that produced it.
